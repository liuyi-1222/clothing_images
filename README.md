# 服装布料换花

把「布料图」上的印花和颜色，换到多张模特图的衣服上。**总体效果**：买家一眼能看出是**同一套衣服**在不同时间/姿势下拍的（同一商品上架图），不是多款相似货。

## 使用

```bash
npm install
npm run dev
```

按页面上的 **1 → 2 → 3 → 4** 步骤操作即可。密钥、接口地址等可在左侧「高级选项」里修改。

- **粘贴图片**：先点一下上传区域，再 `Ctrl+V` / `⌘+V`
- **打包下载**：生成完成后点「打包下载」

## 跨域（仅开发者）

本地若报 CORS，在「高级选项」里把接口地址改为 `http://localhost:5173/t8proxy`（端口以 Vite 输出为准）。

## 安全

密钥只保存在本机浏览器，请勿在公共电脑使用。

## 效果说明

- 布料图决定**花纹和颜色**（底色、印花色相须一致）。只提取布面花色，不会把布料图里的模特带入结果。
- 目标图**无模特**（平铺、挂拍等）时，生成图也**不会出现模特**；有模特时保持同一人，只换布面花色。
- 换另一款布时请先替换第 1 步布料图，并清空目标图后重传，避免和上一款混淆。
- 默认**跟随每张模特图比例**生成（多为 3:4 竖图），可在「高级选项」关闭后手动指定比例。
- 批量生成时各张须**同款同色**；模特脸、发型、姿势须与原图一致。若花纹仍不准，在「补充要求」写明颜色与花纹（如：蓝白碎花、白底），或换模型后重试。
